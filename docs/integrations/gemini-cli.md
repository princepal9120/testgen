# Gemini CLI Integration

Install:

```bash
curl -fsSL https://raw.githubusercontent.com/princepal9120/testgen/main/scripts/install-agent-skill.sh | bash -s -- --agent gemini
```

This writes `GEMINI.md` when the target project does not already have one. If `GEMINI.md` exists, copy the TestGen section manually to avoid clobbering project instructions.
