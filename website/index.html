Review-first workflow

Review first.
Write later.

The safest flow for agents is simple. Generate in dry-run mode, inspect machine-readable artifacts, decide whether the patch is good enough, then materialize files with validation turned on.

1 Generate artifacts with --dry-run --emit-patch --output-format json
2 Inspect results, artifacts, and patches
3 Re-run with --validate when you want files written
