# Support

## Where to Ask Questions

- GitHub Discussions (recommended for usage and design questions)
- GitHub Issues (only for reproducible bugs and feature requests)

## Before Opening an Issue

- Check `README.md` and `docs/CLI_REFERENCE.md`
- Search existing issues and discussions
- Reproduce with the latest version from `main`

## What to Include

- Operating system and version
- TestGen version (`testgen --version`)
- Relevant command and flags
- Minimal reproducible example
- Error output (with secrets removed)

## Commercial or Private Support

No paid support tier is currently offered.
